By the time you've completed this workshop, Trip Planner users should be able to:

Select and set the hotel
Select and add a restaurant
Select and add an activity
Remove the hotel
Remove a restaurant
Remove an activity
Add a day
Remove a day
Switch days
These changes should be reflected in 1) the side panel, and 2) the map. Both should update live as the user edits their plans or switches between days.
